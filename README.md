# py-port-scan (ALPHA version)

py-port-scan is a simple host/ip port scanner.

## Usage

```python
python pyscan.py -h 

usage: pyscan.py [-h] -i IP -p PORT

optional arguments:
  -h, --help            show this help message and exit
  -i IP, --ip IP        host or ip
  -p PORT, --port PORT  port

```
